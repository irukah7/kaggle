# Predict Future Sales
### 概要
このコンペはCourseraコースの「How to win a data science competition」の最終プロジェクトとして実施されている。

このコンペではロシア最大のソフトウェア会社の一つである**1C Company**から提供された毎日の売り上げデータからなる時系列データセットに取り組む。

次の月の全ての商品と店舗の総売上高を予測するのが目的である。
